# Contributing

Thanks for your interest in Raven.

At this stage, Raven is not accepting external pull requests.

The project is still evolving quickly, and keeping the implementation, command surface, and documentation coherent requires a tight review boundary. In practice, that means development is currently maintained by the project owner rather than through community PRs.

Issues are welcome and useful. If you want to contribute, please open an issue for:

- Bugs
- Feature requests
- Documentation problems
- Design feedback

When possible, include:

- Clear reproduction steps
- Expected behavior
- Actual behavior
- Relevant environment details
- A concrete proposal, if you are suggesting a feature or design change

Please check for an existing issue before opening a new one.

If the contribution policy changes in the future, this document will be updated.
